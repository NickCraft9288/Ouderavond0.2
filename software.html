Software

Visual Studio Code

Visual Studio code is het programma dat we momenteel gebruiken om html/css en heel soms ook een beetje Javascript te programmeren.
Visual Studio Code heeft ook handige features zoals Exstensions, met Exstensions kan je het programmeren makkelijker maken en sneller maken doordat je bijvoorbeeld Exstensions kan downloaden die je code verdeeld in kleuren of bijvoorbeeld je code afmaken of sluiten.
Visual Studio Code word ook samen met Github gebruikt om code the delen.

Github

Github gebruiken we om code te delen voor als we bijvoorbeeld een huiswerk opdracht moeten inleveren waarbij we moesten programmeren of wanneer je samen met iemand anders werkt dat je allebij aan dezlefde code kan werken. bij Github gebruiken we vooral momenteel om huiswerk in te leveren, dat kan door naar de website van Github te gaan en daar een reposetory aan te maken en dan al je files daar in te zetten en dan de link van de reposetory inleveren op canvas. Github heeft ook een Pull en Push feature, de Push feature word gebruikt als je samen met iemand anders werkt en je wil je aangespaste code in Github zetten en de Pull is precies het tegenovergestelde de Pull feature gebruik je als je de code die iemand anders heeft gepushed op jouw laptop wil hebben.

Teams

Teams gebruiken we als vooral als we online les hebben elke donderdag maar je kan het ook gebruiken om vragen te stellen aan docenten of om bepaalde bestanden te bekijken. Teams werd helemaal aan het begin van de opleding ook nog gebruikt om opdrachten in te leveren maar dat is uiteindelijk vervangen door canvas. bij Teams kunnen we ook op de kalender zien hoe laat de online lessen zijn en bij assigments kunnen we zien welke opdrachten we moeten inleveren en wanneer we ze moeten inleveren.
