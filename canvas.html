Canvas is een app die we gebruiken om huiswerk in te leveren,
je kan alle vakken zien, als je op een vak klikt en naar opdracht/de week gaat maken de meeste opdrachten moet je via,
github ineveren of via een word document, (github is alleen voor code)
de vakken op canvas zijn: progammeren met html/css, engels, nederlands, ontwerpen, project, en progammeren met javascript
op de kalender kan je zien welke opdrachten op welke dag af moet en in de inbox krijg je je gesprekken te zien.
bij de geschiedenis kan je zien wat je voor het laatste heb opgezocht op die site.
op je account kan je meldingen, profiel en bestanden bekijken.
